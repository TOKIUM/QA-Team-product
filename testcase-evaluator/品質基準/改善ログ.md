# 改善ログ

テストケースの評価・レビュー・生成を通じて得た知見を蓄積します。
定期的にルール化できるものは `記述ルール.md` や `評価基準.md` に昇格させます。

---

## 記録フォーマット

```
### YYYY-MM-DD: [タイトル]
- **発見元**: 評価結果 / レビュー指摘 / 生成時の気付き
- **内容**: 具体的な問題と対応
- **ルール化**: 済 / 未（済の場合は昇格先を記載）
```

---
