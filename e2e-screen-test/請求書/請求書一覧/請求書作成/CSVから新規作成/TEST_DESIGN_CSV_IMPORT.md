## 1. 機能仕様サマリー

### 1-1. 画面フロー

```
請求書一覧 (/invoices)
  └── 「CSVから新規作成」リンク
        └── レイアウト選択画面 (/invoices/import)
              ├── gallery iframe: レイアウトカード一覧（MUI Grid）
              └── カードクリック → CSVインポート画面 (/invoices/import/{ID})
                    └── datatraveler iframe: 4ステップウィザード
                          ├── Step1: CSVファイルアップロード
                          │     └── input[type="file"] → ファイル名表示 → 「項目のマッピングへ」
                          ├── Step2: 項目のマッピング
                          │     └── 帳票データ項目表示 → 「データの確認へ」
                          ├── Step3: データの確認
                          │     └── データ件数表示 → 「帳票プレビューへ」
                          └── Step4: プレビュー → 作成
```

### 1-2. iframe構成

| iframe | name | ドメイン | 用途 |
|--------|------|---------|------|
| gallery | gallery | tpmlyr.dev.components.asaservice.inc | レイアウト選択（MUI Gridカード） |
| datatraveler | datatraveler | tpmlyr.dev.components.asaservice.inc | CSVインポートウィザード全体 |

### 1-3. 既知の制約

| 項目 | 詳細 |
|------|------|
| マッピング問題 | 「前回の設定を利用する」ONでも、キー項目マッピングがundefinedになる |
| クロスオリジン | iframe内はPlaywright `page.frame(name=...)` でのみ操作可能 |
| カスタムDD | マッピング設定のカスタムドロップダウンがPlaywright通常操作で動作しない |
