## 1. 機能仕様サマリー

### 1-1. 画面構成

```
請求書一覧画面 (/invoices)
├── ヘッダー: ロゴ、ユーザー名、ヘルプ、事業所切替
├── サイドバー: 請求書(active)、取引先、帳票レイアウト
├── アクションリンク: 「CSVから新規作成」「PDFを取り込む」
├── 検索フォーム
│   ├── 送付方法（セレクト: 全て/メール/Web送付/郵送代行/FAX送付/その他）
│   ├── 取引先コード / 取引先名 / 自社担当部署 / 自社担当者名
│   ├── 請求書番号 / 合計金額 / ファイル名
│   ├── 承認状況（セレクト: 全て/承認済み/未承認）
│   ├── ステータスCB: 登録中/未送付/送付中/送付済み/送付待ち/登録失敗/送付失敗
│   ├── 請求日（開始日〜終了日）/ 支払期日（開始日〜終了日）
│   └── 「この条件で検索」「リセット」ボタン
├── 一括操作: 請求書を送付する/送付済みにする/その他の操作/選択解除
├── テーブル
│   ├── ヘッダー: □ / 取引先 / 送付先 / ステータス / 承認状況 / 請求書番号 / 合計金額 / 請求日 / 支払期日 / ファイル名
│   └── データ行: クリックで詳細画面に遷移
└── ページネーション: 件数表示、表示件数セレクト
```

### 1-2. 主要操作

| 操作 | 詳細 |
|------|------|
| 検索 | フォーム入力→「この条件で検索」で絞り込み |
| リセット | 「リセット」で検索条件クリア |
| 行クリック | 取引先名セル(td:nth(1))→詳細画面(/invoices/{UUID}) |
| チェックボックス | 個別選択/全選択/解除 |
| 一括操作 | 送付/送付済み/その他の操作（一括添付/一括承認/一括削除） |
