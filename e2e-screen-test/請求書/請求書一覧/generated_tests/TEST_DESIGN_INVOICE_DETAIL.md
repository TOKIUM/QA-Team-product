# 請求書詳細画面 テスト設計書

**対象機能**: TOKIUM 請求書発行 請求書詳細画面
**対象URL**: https://invoicing-staging.keihi.com/invoices/{UUID}
**作成日**: 2026-02-18
**テスト項目数**: 15件（自動15件）

---

## 1. 機能仕様サマリー

### 1-1. 画面構成

```
請求書詳細画面 (/invoices/{UUID})
├── ナビゲーション
│   ├── 戻るボタン → /invoices（一覧）に戻る
│   ├── パンくず: 請求書 > 帳票情報
│   └── ページ送り: 「前の請求書」「次の請求書」+ 位置表示（"1 / 8013件"）
├── ステータス表示: 未送付 / 送付済み / 送付中
├── アクションボタン: 送付済みにする / 承認する / 削除
├── タブ
│   ├── 帳票情報（デフォルト）
│   │   ├── 取引先情報: 取引先コード、取引先名、送付先担当者情報、自社担当者情報、「取引先を選択する」ボタン
│   │   ├── 送付先情報: 送付方法
│   │   ├── 帳票項目: 合計金額、請求日、支払期日、請求書番号
│   │   ├── 基本情報: 管理ID、登録日時、登録方法、登録者
│   │   └── メモ: textbox + 「メモを保存する」ボタン
│   └── 添付ファイル（?tab=attachment）
│       ├── アップロード: 「ファイルを選択」ボタン、D&Dエリア
│       └── 使用状況: 添付件数、ファイルサイズ
└── 取引先選択ダイアログ: 検索ボタン、現在の取引先表示、閉じるボタン
```

### 1-2. 画面遷移

| 遷移元/先 | 操作 | 遷移先 |
|-----------|------|--------|
| 一覧→詳細 | テーブル行td:nth(1)クリック | /invoices/{UUID} |
| 詳細→一覧 | 戻るボタン | /invoices |
| 詳細→一覧 | パンくず「請求書」リンク | /invoices |
| 詳細→次の詳細 | 「次の請求書」ボタン | 同一URL、位置更新 |
| 詳細→前の詳細 | 「前の請求書」ボタン | 同一URL、位置更新 |
| 帳票情報→添付 | 「添付ファイル」タブクリック | ?tab=attachment付与 |
