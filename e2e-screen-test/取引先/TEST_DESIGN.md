## 1. 機能仕様サマリー

### 1-1. 画面構成

```
取引先一覧画面 (/partners)
├── ヘッダー: ロゴ、ユーザー名、ヘルプ、事業所切替
├── サイドバー: 請求書、取引先(active)、帳票レイアウト
├── アクションボタン: 「取引先を追加」「その他の操作」
│   └── その他の操作メニュー: 取引先インポート / 取引先エクスポート
├── 検索フォーム（折りたたみ式）
│   ├── 送付方法（セレクト: email等）
│   ├── 取引先コード / 取引先名 / 送付先担当者名 / 自社担当者名 / メールアドレス / 識別コード
│   ├── CB: 凍結された取引先を含む / Web送付設定の変更要求がある
│   └── 「この条件で検索」「リセット」ボタン
├── テーブル
│   ├── ヘッダー: 取引先コード / 取引先名 / 送付先 / 送付先担当者名 / 自社担当者名 / 登録日 / 識別コード
│   └── データ行: クリックで更新モーダル表示
├── ページネーション: 件数表示、表示件数セレクタ（20/100等）、前後ページ
├── 更新モーダル（取引先更新）
│   ├── 取引先項目タブ: コード/名称/送付方法/住所/送付先担当者/自社担当者/基本情報/識別コード
│   ├── Web送付設定タブ: 閲覧用メールアドレス/新規メールアドレス追加
│   └── ボタン: 保存 / キャンセル / 凍結する / ×ボタン
└── 新規追加モーダル（取引先入力）
    ├── コード/名称/送付方法(デフォルト:メール)/送付先担当者/送付先メールアドレス/自社担当者
    └── ボタン: 保存 / キャンセル（凍結ボタンなし）
```

### 1-2. 検索フォームの特徴

| 特徴 | 詳細 |
|------|------|
| 折りたたみ | 「検索条件」ボタンで展開/閉じる |
| インターセプト回避 | click(force=True) が必要な場合あり |
| フォーム閉じ必須 | 展開中はテーブル行クリックがインターセプトされる |
